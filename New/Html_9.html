Introduction to array

What is an array?

An array is a collection of items, or data, arranged in rows and columns. It can be used to store multiple pieces of data of the same type together.

Characteristics of array

  • Each element in a character array is a single character, usually represented by the 'char' data type in programming languages.
  • Accessing characters within the array is done using an index number, starting from 0.
  • Character arrays are commonly used to represent strings in programming languages where there is no dedicated string data type, with the null terminator ('\0') signifying the end of the string.
  • Like most arrays, character arrays store their data in contiguous memory locations, allowing for efficient access to elements.


Types of Arrays in Data Structure

  • One-Dimensional Arrays (1D Arrays) These are the simplest form of arrays, consisting of a single line of elements
    • An array of integers with 5 elements can be declared and initialized as int numbers[5]
  • Two-Dimensional Arrays (2D Arrays) A two-dimensional array in data structure, often thought of as a matrix, consists of rows and columns.
    • an array that is organized in rows and columns.
  • Multi-Dimensional Arrays.

Tree


what is Tree?


a tree is a non-linear hierarchical data structure that consists of nodes connected by edges



Key Points:


  • Nodes: Each node in the tree contains data and can have zero or more child nodes.
  • Edges: Edges represent the connections between nodes.
  • Root: The topmost node in the tree is called the root.
  • Parent: A node that directly connects to another node below it is called the parent node.
  • Child: A node that is directly connected to a node above it is called the child node
  • Leaf: A node that has no children is called a leaf node.


Common Types of Trees:

  • Binary Tree:
    • A tree in which each node can have at most two children.
  • Binary Search Tree (BST):
    • A binary tree where the left child of a node contains values less than the node's value, and the right child contains values greater than the node's value.
  • AVL Tree
    • A self-balancing BST that maintains a height difference of at most 1 between its left and right subtrees.
  • Red-Black Tree
  • Another type of self-balancing BST that maintains a specific set of rules to ensure balance


Implementing Trees in C++:

You can implement trees in C++ using classes and pointers. Here's a simple example of a binary tree node:C++

            #include <iostream>

                struct Node {
                    int data;
                    Node* left;
                    Node* right;
                
                    Node(int val) : data(val), left(nullptr), right(nullptr) {}
                };


Applications of Trees

  • File systems:
    • Trees can represent the hierarchical structure of a file system.
  • Expression parsing:
    • Trees can be used to represent and evaluate mathematical expressions.
  • Search algorithms:
    • Trees can be used to implement efficient search algorithms, such as binary search
  • Decision trees:
    • Trees can be used to model decision-making processes in machine learning.

Graph

What is graph?

a graph is a non-linear data structure that represents a collection of nodes (also called vertices) and edges that connect these nodes. Graphs are used to model relationships between objects, where the nodes represent the objects, and the edges represent the connections or associations between them.

Components of a Graph:

  • Vertices:
    • The fundamental units of a graph, representing objects or entities.
  • Edges:
    • The connections between vertices, representing relationships or associations.
  • Weighted Graph:
    • A graph where each edge has an associated weight or cost.


Representing Graphs in C++:


There are two common ways to represent graphs in C++:


  • Adjacency Matrix:
    • A 2D array where the rows and columns represent the vertices. The value at matrix[i][j] indicates whether there is an edge between vertex i and vertex j.
  • Adjacency List:
    • An array of lists, where each list represents the neighbors of a particular vertex.


            #include <iostream>
                #include <vector>
                #include <list>
                
                using namespace std;
                
                class Graph {
                private:
                    int V; // Number of vertices
                    vector> adj; // Adjacency list
                
                public:
                    Graph(int V) : V(V), adj(V) {} // Constructor
                
                    void addEdge(int u, int v) {
                        adj[u].push_back(v); // Add v to u's list
                        adj[v].push_back(u); // Add u to v's list (for undirected graph)
                    }
                
                    void printGraph() {
                        for (int i = 0; i < V; ++i) {
                            cout << "Adjacency list of vertex " << i << ": ";
                            for (auto it = adj[i].begin(); it != adj[i].end(); ++it) {
                                cout << *it << " ";
                            }
                            cout << endl;
                        }
                    }
                };
                
                int main() {
                    // Create a graph with 5 vertices
                    Graph g(5);
                
                    // Add edges
                    g.addEdge(0, 1);
                    g.addEdge(0, 4);
                    g.addEdge(1, 2);
                    g.addEdge(1, 3);
                    g.addEdge(1, 4);
                    g.addEdge(2, 3);
                    g.addEdge(3, 4);
                
                    // Print the adjacency list representation
                    g.printGraph();
                
                    return 0;
                }
